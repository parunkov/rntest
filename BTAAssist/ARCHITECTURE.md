# Архитектура проекта BTA Assist

## Обзор

BTA Assist - это мобильное приложение на React Native Expo с Pixel Perfect версткой по макету Figma для медицинской клиники.

## Структура проекта

### `/src/components` - Переиспользуемые компоненты

#### `AppointmentCard.tsx`
Карточка приема пациента.

**Props:**
- `appointment: Appointment` - данные о приеме

**Особенности:**
- Размеры: 340x65px
- Border radius: 18px
- Аватар: 55x55px с border-radius 14px
- Цветовая индикация типа приема

#### `ServiceCard.tsx`
Карточка услуги/сервиса.

**Props:**
- `icon: React.ReactNode` - SVG иконка
- `title: string` - название услуги
- `onPress?: () => void` - обработчик нажатия

**Особенности:**
- Высота: 101px (ширина адаптивная)
- Ширина: рассчитывается динамически - (ширина экрана - отступы - gap) / 3
- Border radius: 18px
- Иконка: 36x36px
- Адаптивная верстка: всегда 3 карточки в ряд на любой ширине экрана

#### `BottomNav.tsx`
Нижняя навигация.

**Особенности:**
- Высота: 54px
- 5 иконок навигации
- Центральная кнопка (50x50px) с эффектом elevation
- Бейдж с уведомлениями (21x21px)

#### `Icons.tsx`
Библиотека SVG иконок.

**Компоненты:**
- `ArrowIcon` - стрелка назад
- `BotoxIcon` - иконка с градиентом
- `BlepharoplastyIcon` - иконка блефаропластики
- `FaceIcon` - иконка лица
- `RepeatIcon` - иконка повторного приема
- `CalendarIcon` - иконка календаря
- `FolderIcon` - иконка папки
- `LearningIcon` - иконка учебника
- `PendingIcon` - иконка ожидания
- `HomeIcon`, `CalendarNavIcon`, `SearchIcon`, `ProfileIcon` - иконки навигации
- `PlusIcon` - центральная кнопка

Все иконки с градиентами используют одинаковую цветовую схему (аквамарин #56B5B3 → #1F7876).

### `/src/screens` - Экраны приложения

#### `HomeScreen.tsx`
Главный экран приложения.

**Элементы:**
1. Шапка с заголовком и кнопкой назад
2. Сетка карточек услуг (6 карточек, 3 колонки)
3. Заголовок "Ближайшие приемы" с кнопкой "Архив"
4. Список карточек приемов
5. Нижняя навигация

**Размеры:**
- Экран: 360x720px
- Отступы: 10px по бокам
- Gap в сетке: 10px

### `/src/constants` - Константы

#### `colors.ts`
Цветовая палитра приложения, извлеченная из Figma.

#### `fonts.ts`
Шрифты и размеры шрифтов.

### `/src/types` - TypeScript типы

#### `index.ts`
Определения типов данных.

**Типы:**
- `Appointment` - данные о приеме пациента

## Дизайн-система

### Цвета
```typescript
background: '#F6F6F6'    // Фон приложения
white: '#FFFFFF'         // Белый (карточки)
text.primary: '#222221'  // Основной текст
text.secondary: '#616161' // Второстепенный текст
accent.red: '#D7131F'    // Красный (бейдж)
accent.blue: '#1BC4EA'   // Голубой (повторный прием)
accent.green: '#009C6A'  // Зеленый (первичный прием)
accent.darkBlue: '#026FD5' // Темно-синий (осмотр)
accent.teal: '#1F7876'   // Бирюзовый (кнопки)
```

### Градиент (Аквамарин)
```
linear-gradient(180deg, #56B5B3 0%, #1F7876 100%)
```

### Шрифты
- **Семейство:** Montserrat
- **Начертания:** Regular (400), Medium (500), SemiBold (600)
- **Размеры:** 10px, 12px, 14px, 16px, 18px

### Радиусы скругления
- Карточки услуг: 18px
- Карточки приемов: 18px
- Аватары: 14px
- Навигация: 12px
- Бейдж: 99px (круг)

### Отступы и размеры
- Ширина экрана: 360px
- Высота экрана: 720px
- Отступы по бокам: 10px
- Gap между элементами: 10px
- Высота шапки: 60px
- Высота навигации: 54px

## Pixel Perfect

Все размеры, отступы, цвета и шрифты точно соответствуют макету Figma:
- Размеры элементов скопированы с точностью до пикселя
- Цвета взяты из дизайна (HEX коды)
- Шрифты и их размеры идентичны макету
- Letter spacing и line height соответствуют дизайну

## Производительность

- SVG иконки оптимизированы и созданы как React компоненты
- Градиенты переиспользуются через компоненты
- Минимальное количество re-renders
- Оптимизированные изображения

## Расширяемость

Проект легко расширяется:
1. Новые экраны добавляются в `/src/screens`
2. Новые компоненты в `/src/components`
3. Новые иконки в `Icons.tsx`
4. Новые типы в `/src/types`
